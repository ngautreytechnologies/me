## Project Goals

This CV site is intentionally built without heavy frontend frameworks.

Instead, it demonstrates the same principles I apply in larger platform work:

- clear separation of concerns
- modular system boundaries
- maintainable evolution over time
- simple, deterministic execution

The result is a CV that is both human-readable and architecturally disciplined.
